# Enable GitHub Pages

## Quick Steps:

1. Go to: https://github.com/RG7on/fl-monitoring-dashboard/settings/pages

2. Under "Source", select **main** branch

3. Click **Save**

4. Wait 1-2 minutes

5. Your dashboard will be live at:
   **https://rg7on.github.io/fl-monitoring-dashboard/dashboard.html**
